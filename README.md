# PruebaTecnica

-Ejecución-

1. Clonar el repositorio o descargar el proyecto.
2. Posicionarse en /PruebaTecnica
3. Ejecutar el archivo principal:

python main.py

-----

-Uso-

1. Seleccionar la opción para cargar un archivo CSV. (Opción 1)
2. Ingresar el nombre del archivo INCLUYENDO LA EXTENSIÓN. Ejemplo:

books.csv
libros.csv
   
3. Navegar por las opciones del menú para:

Guardar libros en base de datos
Buscar libros por autor o año
Listar libros
Agrupar información

4. Después de mostrar cada resultado, presione ENTER para continuar.

-----

-Funcionalidades-

Lectura de archivos CSV
Limpieza y normalización de datos
Eliminación de duplicados
Búsqueda por:

Autor
Año de publicación

Agrupación de libros:

Por autor
Por año si el autor es desconocido

-----

El archivo CSV debe tener el siguiente formato:

Title,Author,Publication Year
